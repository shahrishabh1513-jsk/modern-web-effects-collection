## 🌟 About The Collection

**Modern CSS Effects Collection** is a frontend showcase project featuring 5 independent mini-projects focused on creative design, smooth animations, and interactive user experiences.

Built using pure **HTML, CSS, and JavaScript**, this collection demonstrates advanced styling techniques, animation systems, UI customization, and responsive design patterns.

Perfect for portfolio demonstration and CSS mastery practice.

---

# 📋 Projects Overview

---

## 1️⃣ ⏰ Modern Digital Clock & Stopwatch

A beautifully designed real-time digital clock combined with a fully functional stopwatch system.

### ✨ Features
- Real-time digital clock
- AM / PM indicator
- Dynamic greeting message
- Current date display
- Stopwatch with lap timing
- Start / Pause / Reset controls
- Dark / Light theme toggle
- Smooth glassmorphism UI

---

## 2️⃣ 🎨 Creative Loading Animations

A collection of 8 unique loading animations designed to enhance user experience during content loading.

### ✨ Features
- Pulsing dots animation
- Spinning rings loader
- Bouncing balls effect
- Gradient progress loader
- Neon pulse animation
- 3D rotating cube
- Ripple wave effect
- Animated loading text

Perfect for websites, dashboards, and applications.

## 3️⃣ 📜 Modern Custom Scrollbar Showcase

- Live style switcher
- Width adjustment slider
- Scroll progress indicator
- Dark / Light theme toggle
- Smooth transition effects

Demonstrates advanced CSS pseudo-element styling and UI customization.
